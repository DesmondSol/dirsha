# üå± Dirsha - Agricultural Investment & Farming Platform

Dirsha is a dual-platform ecosystem connecting agricultural investors with farmers through a secure bonds marketplace and comprehensive farming support system. The platform consists of:

1. Investor Web Platform - A powerful dashboard for investors to browse, analyze, and invest in agricultural bonds
2. Farmer Mobile App - A mobile application providing farmers with market data, crop advice, and access to investment opportunities

[![Dirsha Mobile Demo](https://img.shields.io/badge/Farmer%20Platform-Available-brightgreen)](https://dirsha.vercel.app/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## üåü Platform Overview

### For Investors (Web Platform)
- Bonds Marketplace: Browse and invest in agricultural bonds with transparent terms and projected returns
- Portfolio Management: Track your investments, returns, and farmer progress in real-time
- Risk Analysis: Access detailed risk assessments and performance metrics for each investment
- Document Management: Securely store and manage all investment documents and contracts
- Farmer Profiles: View detailed profiles of farmers and their agricultural projects

### For Farmers (Mobile App)
- Market Prices & Trends: Access real-time agricultural commodity prices
- Bonds Marketplace: Connect with investors and secure funding through fair contracts
- Crop-Specific Advice: Get tailored recommendations for planting and cultivation
- Multilingual Support: Use the platform in multiple languages including English and Amharic
- Voice Commands: Navigate and interact with the platform using voice
- Push Notifications: Get instant updates on price changes and market trends
- Location Services: Get localized agricultural information and nearby market prices

## üõ† Getting Started

### For Investors (Web Platform)

#### Prerequisites
- Node.js (v14 or later)
- npm (v7 or later)

#### Setup Instructions

1. Clone the repository:
      git clone https://github.com/DesmondSol/dirsha.git
   cd dirsha
   

2. Install dependencies:
      npm install
   

3. Set up environment variables:
   Create a .env.local file in the root directory with required variables:
   
   GEMINI_API_KEY=your_gemini_api_key
   NEXT_PUBLIC_API_URL=https://api.dirsha.com
   # Add other environment variables as needed
   

4. Run the development server:
      npm run dev
   

5. Access the platform at [http://localhost:3000](http://localhost:3000)

### For Farmers (Mobile App)

#### Prerequisites

## üèó Technology Stack

### Investor Web Platform
- Frontend Framework: [Next.js](https://nextjs.org/) - React framework for server-rendered applications
- Data Visualization: [Recharts](https://recharts.org/) - Interactive charts for investment analytics
- Form Handling: [React Hook Form](https://react-hook-form.com/) - For efficient form validation and submission
- API Client: [Axios](https://axios-http.com/) - For making HTTP requests
- Authentication: [NextAuth.js](https://next-auth.js.org/) - For secure authentication

- ### Farmer Mobile App
- Framework: [React Native](https://reactnative.dev/) - For cross-platform mobile development
- State Management: [Redux](https://redux.js.org/) - Centralized state management
- Navigation: [React Navigation](https://reactnavigation.org/) - For handling in-app navigation
- UI Components: [React Native Paper](https://reactnativepaper.com/) - Material Design components
- Icons: [React Native Vector Icons](https://github.com/oblador/react-native-vector-icons) - Customizable icons
- Offline Support: [Redux Persist](https://github.com/rt2zz/redux-persist) - For offline data persistence
- Maps & Location: [react-native-maps](https://github.com/react-native-maps/react-native-maps) - For location-based features
